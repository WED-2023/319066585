    • about me


    סבתא ורדה נולדה בשנת 1938 בחצי האי ג'רבה, טוניס. כשהייתה צעירה, בשנת 1951, עלתה לארץ באונייה והייתה בין הראשונים שהתיישבו במושב בית הגדי שבצפון הנגב

    עם עלייתה לארץ, עבדה בחקלאות בקיבוץ סעד, ושם פגשה את בעלה פרז'י אליהו, סבא שלי. יחד הקימו משפחה וברבות השנים נולדו להם עשרה ילדים

    במשך השנים ניהלו משק חקלאי וגידלו בו פרות, תרנגולות, פרחים ועוד. סבתא ורדה הייתה אישה שמחה, מלאה באנרגיה חיובית, וידועה במיוחד בהכנסת האורחים החמה שלה – דלת ביתה הייתה תמיד פתוחה לכולם

    בנוסף, הייתה בשלנית מדהימה, והכינה את המאכלים המסורתיים שלה בטעמים ייחודיים שאי אפשר לשחזר

    כולם כינו אותה כמלכת הדומינו, וכל בני המשפחה למדו ממנה את סודות המשחק

    שלושה הבדלים מרכזיים בין גרסאות ה-Web:

    אופי האינטראקציה עם התוכן

    Web 1

    המשתמשים היו בעיקר צרכני מידע ללא יכולת לתרום תוכן או להגיב. אתרים היו סטטיים וכללו בעיקר טקסט ותמונות פשוטות

    Web 2

    האינטרנט הפך לאינטראקטיבי – משתמשים יכלו ליצור, לשתף ולהגיב לתוכן, מה שהוביל לצמיחת רשתות חברתיות, בלוגים ופורומים

    Web 3

    מרחיב את היכולות האינטראקטיביות בעזרת טכנולוגיות מתקדמות כמו בינה מלאכותית ובלוקצ’יין, ומאפשר חוויית שימוש מותאמת אישית לצד אינטגרציה של מכשירים חכמים


    השימוש בטכנולוגיות

    Web 1

    הסתמך בעיקר על HTML סטטי, ללא אינטראקציות מתקדמות

    Web 2

    נוספו טכנולוגיות כמו JavaScript שאיפשרו יצירת אתרים דינמיים עם תגובות בזמן אמת

    Web 3

    מבוסס על שילוב של AI, למידת מכונה ובלוקצ'יין, ומאפשר למערכות להתאים את המידע באופן חכם ואוטומטי לכל משתמש


    שליטה על הנתונים

    Web 1

    כל המידע היה בבעלות בעלי האתרים, והמשתמשים היו צרכנים פסיביים

    Web 2

    פלטפורמות גדולות החלו לאסוף מידע רב על המשתמשים ולנתח את פעילותם לצרכים מסחריים

    Web 3

    משנה את המודל הזה בעזרת טכנולוגיות מבוזרות, כך שהשליטה על המידע חוזרת למשתמשים, והם מחליטים מי יכול לגשת אליו וכיצד להשתמש בו

    אינטראקציה עם התוכן

    Web 1

    משתמש שמחפש סרט מומלץ לצפייה ימצא עמוד טקסט פשוט עם המלצות של כותב האתר, ללא אפשרות להגיב, לשתף או לקבל הצעות נוספות

    Web 2

    המשתמש יכול למצוא המלצות על סרטים בבלוג או באתר המלצות לצפייה שמאפשר הוספת תגובות משתמשים, דירוגים ושיתוף ברשתות חברתיות

    Web 3

    מאפשר חוויית חיפוש חכמה יותר – מערכת מבוססת בינה מלאכותית יכולה להציע למשתמש סרט מותאם אישית לפי העדפותיו, היסטוריית צפייה ודפוסי חיפוש קודמים


    טכנולוגיות

    Web 1

    מציג את ההמלצה על הסרט בדף HTML פשוט, ללא עיצוב מתקדם או אינטראקטיביות

    Web 2

    האתרים משתמשים ב-CSS ובעיצוב דינמי, ומאפשרים גם הצגת תמונות, סרטונים ותגובות משתמשים

    Web 3

    המערכת יכולה במקרים מסויימים להתחבר למסכים חכמים בבית, לדוגמה – לשלוח את סרטים המומלצים ישירות לטלוויזיה או להציע סרטים נוספים למשתמש שהוא לא ידע על קיומם


    שליטה על הנתונים

    Web 1

    המידע באתר נשאר קבוע וללא איסוף נתונים על המשתמשים

    Web 2

    מנועי חיפוש ואתרי המלצות על סרטים אוספים מידע על הרגלי החיפוש של המשתמשים כדי להציע להם תוכן מותאם אישית, אך גם לצרכים פרסומיים

    Web 3

    מאפשר למשתמשים לשמור על פרטיותם ולשלוט במידע שלהם, כך שהם יכולים לבחור אם לשתף נתונים מסוימים או לקבל המלצות באופן אנונימי
